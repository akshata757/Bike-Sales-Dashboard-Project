1. Project Overview
   
This Excel dashboard is designed to help analyze the factors that influence bike purchases. By combining demographic attributes, income levels, and commute-related variables, the dashboard allows users to identify trends and patterns across different customer groups.
The project integrates fully prepared data, pivot tables, and professionally styled visuals to support business decision-making.

--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

2. Dashboard Features

The dashboard provides the following capabilities:
- Dynamic visualizations using pivot charts  
- Fully interactive filtering using slicers  
- Structured layout for simple navigation  
- Segmentation based on gender, marital status, education, and region  
- Comparison between customers who purchased a bike and those who did not

--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

3. Key Visual Insights
   
Average Income per Purchase
A bar chart showing the difference in average income between customers who purchased a bike and those who did not, segmented by gender. This helps identify income patterns that contribute to purchase decisions.

Customer Age Brackets
A line chart analyzing purchase counts across age groups such as:
- Adolescent  
- Middle Age  
- Old  
This visualization highlights which age brackets show higher purchasing likelihood.

Customer Commute Analysis
A line chart comparing bike purchases across commute distances:
- 0–1 Miles  
- 1–2 Miles  
- 2–5 Miles  
- 5–10 Miles  
- More than 10 Miles  
This helps understand how travel distance influences bike-buying behavior.

--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

4. Interactive Filters

The dashboard includes slicers that allow users to segment data dynamically:
- Marital Status  
- Region  
- Education  
- Gender  
These filters update all charts simultaneously, providing flexible and targeted analysis.

--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

5. Dataset Structure

The dataset contains the following fields:
- Age and age brackets  
- Gender  
- Marital status  
- Education level  
- Region (Europe, North America, Pacific)  
- Commute distance  
- Income  
- Bike purchase indicator (Yes / No)  
- Additional household and demographic attributes
All fields are cleaned, structured, and optimized for pivot-based analysis.

7. How to Use

Download or clone the repository.  
Open the file named `Bike\_Sales\_Dashboard\_Project.xlsx`.  
Navigate to the Dashboard sheet.  
Use slicers on the left side to filter customer segments.  
Review how each visualization updates based on your selections.  
Explore or modify the dataset as needed for extended analysis.
